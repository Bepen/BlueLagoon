# BlueLagoon


## How To Run
Pull down repo and navigate to root. Start the server and client app in separate terminals.

To start server:
```bash
npm start # this starts a server on port 3001
```

To start client:
```bash
cd client
npm start

Navigate to http://localhost:3000
```
